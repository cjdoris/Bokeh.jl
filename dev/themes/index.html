Themes

See the theming section to find out how to apply these themes.

caliber

using Bokeh
p = figure(title="caliber", width=300, height=300)
plot!(p, Line; x=[1, 2, 3, 4, 5], y=[6, 7, 6, 4, 5])
Document(p, theme="caliber")
